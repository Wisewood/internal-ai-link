# internal-ai-link

WitAI chat frontend served at [ai.wisewoodint.com](https://ai.wisewoodint.com).

## Stack

- Vite
- TypeScript
- React
- shadcn/ui
- Tailwind CSS
- Supabase

## Local development

Requires Node.js and npm.

```sh
npm install
npm run dev
```

Dev server runs on port 8080.

## Build

```sh
npm run build
```
